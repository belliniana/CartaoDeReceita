Projeto – Cards de Receitas com HTML e CSS

Este projeto apresenta uma coleção de cards de receitas estilizados, organizados em um layout moderno e totalmente responsivo. Cada card destaca uma imagem, título e breve descrição da receita, proporcionando uma navegação visual agradável e intuitiva.
O objetivo principal é praticar organização visual, uso de flexbox, responsividade e estilização com gradientes.

Sobre o Projeto

A proposta deste projeto é criar um catálogo visual inspirado em sites de culinária, onde cada receita é exibida como um card independente.
Foram combinados HTML semântico e CSS moderno para construir uma interface leve, estética e funcional.

Os cards contam com:

Imagens de alta qualidade exibidas com object-fit: cover;

Títulos e descrições curtas e objetivas;

Efeitos de hover com transição suave;

Gradientes suaves para reforçar o visual elegante.

A estrutura utiliza Flexbox para organização dos cards, permitindo que eles se alinhem e se adaptem automaticamente conforme o tamanho da tela. Também foram aplicadas media queries para ajustar larguras em tablets e smartphones.

Principais Recursos e Técnicas Utilizadas
HTML5 Semântico

Uso de classes bem definidas (recipe-container, recipe-card, recipe-content).

Organização visual e estrutural limpa e legível.

CSS3 Avançado

Flexbox:
Organização flexível dos cards, com espaçamento automático e adaptação responsiva.

Responsividade com Media Queries:
Ajuste automático do tamanho dos cards em telas menores (tablets e celulares).

Gradientes Lineares:
Aplicados tanto ao background principal quanto aos cards para criar profundidade visual.

Efeitos de Hover e Transições:
Animações suaves ao passar o mouse, com destaque no card e mudança de iluminação.

Sombras e Bordas Decorativas:
Para ressaltar cada card e dar aparência mais profissional.

Design

Paleta de cores suaves e aconchegantes.

Fotos recortadas proporcionalmente mantendo o foco no alimento.

Layout inspirado em catálogos gastronômicos.

Como Executar

Clone este repositório ou faça o download dos arquivos.

Abra o arquivo index.html diretamente no navegador.

Certifique-se de que a pasta assets está no mesmo diretório, contendo imagens e o arquivo CSS.

Finalidade do Projeto

Este projeto foi criado com foco em prática e aprimoramento de:

Organização visual com Flexbox

Estruturação de múltiplos componentes (cards)

Uso de gradientes e sombras

Responsividade CSS

Montagem de um layout temático e atraente

É uma excelente base para evoluir para um catálogo real, uma página de receitas ou até mesmo um sistema dinâmico no futuro.
